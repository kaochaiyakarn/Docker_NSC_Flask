How to use

Sleep Stage Scoring



0. Click "Upload files" on the menu above the website.

1. Select option.

Create new model -> Creating new model from your own files

Predict result -> To predict sleep stage from user input signal.

*If user want to create new model, look at 2)


**If user want to predict sleep stage, look at 3)



2. Create new model

2.1 Select model from public dataset

- Software has 3 model from public dataset to let user
choose and try which one is suitable for your data,
which are Sleep-EDF, ISRUC-SLEEP, MASS model.

2.2 Upload signal file

- Uploading signal file as user want.
* Signal file is a file that contain sleep signal of subject, and the file extension is .edf
* For example, Sleep01.edf

2.3 Upload label file

- Uploading label file as user want.
* Label file is a file that contain sleep stage result of subject, and the file extension is .edf
* For example, Label01.edf
* Label file need to match with signal file that user already upload in 2.2

2.4 Click at Create new model to finished process

2.5 Wait for the new model

* User can see the estimate training time from Estimate training time
When already finished process in 2.4)

* Or user can see the estimate training time from menu above the browser

Estimate training time

3. Predict Signal

3.1 Select model from public dataset ( SLEEP-EDF, ISRUC-SLEEP, MASS )

from user created in 2)

3.2 Select signal file that user want to predict result.

3.3 Click on Predict Result to finished process.

3.4 See the prediction result from click Show Results when user finished 3.3)

or click Results In menu bar above the website.
